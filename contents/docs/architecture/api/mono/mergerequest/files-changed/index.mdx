---
title: Files Changed
description: Use Libra Commend to get mr's changed files.
---

**URL:**

    ```bash
    GET /api/v1/mr/${link}/files-changed
    ```

**Request Params:**

| **Field** | **Required** | **Type** | **Location** | **Description**                                        |
| --------- | ------------ | -------- | ------------ | ------------------------------------------------------ |
| link      | Y            | string   | Path         | A unique identifier for the merge request to be merged |


**Response Params:**
1. Common

| **Field** | **Type** | **Description**    |
| --------- | -------- | ------------------ |
| content   | string   | libra diff string  |
| files     | array    | changed files list |

2. files list

| **Field**       | **Type**         | **Description**                  |
| --------------- | ---------------- | -------------------------------- |
| path            | string           | The generated MR link            |
| status          | string           | The status of changed file  <br/> - "new": New added files. <br/> - "modified": Changed files.<br/> - "deleted": Removed files          |




**Request Example:**


```json
GET /api/v1/mr/OTLRROOS/files-changed
Content-Type: application/json
```

**Response Example:**:
```json
{
```
