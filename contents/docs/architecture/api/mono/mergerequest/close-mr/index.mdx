---
title: Close MR
description: Close a Merge Request (MR) if it is in an open state
---

**URL:**

    ```bash
    POST /api/v1/mr/${link}/close
    ```

**Request Params:**

| **Field** | **Required** | **Type** | **Location** | **Description**                                        |
| --------- | ------------ | -------- | ------------ | ------------------------------------------------------ |
| link      | Y            | string   | Path         | A unique identifier for the merge request to be merged |
| Cookie    | Y            | string   | Header       | Cookie of logged-in user                               |

**Request Example:**

```json
POST /api/v1/mr/OTLRROOS/close
Content-Type: application/json
```
